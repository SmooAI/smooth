Headline finding: direct mode lets the operator solve in ~3 minutes

Cell D — direct mode + smooth-fast-gemini Woz — finished in 3.7 minutes. The operator (Kimi K2-Thinking via Moonshot) was done in 3 minutes because:

  • No microVM boot (~30-60s saved)
  • No sandbox-internal tool installs
  • Native filesystem and CPU (no virtualization tax)
  • Workspace files visible to host directly — no bind-mount roundtrips

Cell A's 16.8–20.2 min and Cell C's 32.2 min reflect the sandbox tax. Cell B's 23.7 min in direct mode was inflated by Big Smooth's Kimi-BS hitting the 900s chat-turn ceiling on tick #3 — the operator was actually done at ~3 min like cell D, but the bench's deadline polling waited for the supervisor's long tick to return.

What it means for production use

DecisionRecommendation
Default dispatch mode for bench & CI? Direct (no sandbox) when running on trusted code (your own repos, polyglot bench). 5-10× wall-time win. Use sandbox when running untrusted code or when you need Narc/Wonk/Goalie audit.
Default Woz model? Either works. gemini-3.1-flash-lite-preview (direct native pass-through) for explicit Gemini control; smooth-fast-gemini (LiteLLM alias) for budget/observability via the gateway. They route to the same upstream (Gemini 2.5 Flash Lite). Pick the alias by default for cost reporting.
Default chat-agent model? Sandbox: smooth-reasoning-kimi as set today. Direct mode reveals Kimi-BS as the dominant cost on later ticks — its 900s stall on cell B's tick #3 wiped out the direct-mode speedup. Worth experimenting with claude-haiku-4-5 or smooth-fast-gemini for the chat-agent slot specifically when you want fast coaching feedback.

Caveats

  • Single task × single operator-LLM — affine-cipher is small. Harder tasks (rust/alphametics, java/alphametics) where take 9 saw regressions will likely show wider matrix variance.
  • n=1 per cell except A (n=3). The trends are clear but individual cell wall times have variance. D's 3.7 min in particular needs re-running to confirm it's reproducible (cell A's variance was 16.8–20.2 min).
  • Cost = $0 across all cells — gateway pricing for Moonshot/Z.ai/MiniMax was missing. Fix shipped via the smooai monorepo's litellm-moonshot-pricing branch (merged to main); production deploy was in flight at write time.
  • Direct mode caveat: operator runs as a regular host process, no Narc/Wonk/Goalie. Fine for trusted task content (your own repos, public benchmarks); not safe for untrusted code from external sources.
